Keep this page secret. It’s not linked in the site and is blocked from search engines. Access directly at /admin-notes.html.

1) Update Prices

Open script.js and edit the PRICES object. Labels must contain a number so Paystack can read the amount:

MTN: [
  {{ label: '1GB — GHS 5' }},
  ...
],
Netflix: [
  {{ label: 'Monthly Subscription — GHS 45' }}
],
Audiomack: [
  {{ label: 'One-time Subscription — GHS 30' }}
]

Note: The number in the label is parsed and sent to Paystack as GHS in pesewas.

2) Paystack

Public key is set near the top of script.js:

const PAYSTACK_PUBLIC_KEY = "pk_live_...";

Change it if needed. Currency is already set to GHS.

3) Contact Numbers

In script.js:

const PHONE = "0534706233";
const SMS_NUMBER = "0534706233";

4) What Each Button Does

  • Pay Now (Data): Reads selected plan + phone + email → opens Paystack with amount and metadata.
  • Pay Netflix / Pay Audiomack: Reads plan + email → opens Paystack.

5) Troubleshooting

  • If the Paystack popup doesn’t show, verify internet connection and that the script https://js.paystack.co/v1/inline.js loads.
  • If amount is 0 or wrong, ensure the plan label includes a number (e.g., GHS 45).
  • For AFA/Voucher manual charges (if added), use the prompt to enter the GHS amount.

6) Deployment Tips

  • Use GitHub Pages: Settings → Pages → Build from branch (main).
  • For Netlify: drag-drop the folder. No build step needed.
